## 为什么有这本手册

大模型部署听起来简单——`pip install` + `app.py` = 完成。实际操作中会遇到：

- 层数限制、超时、apt 源不可达
- CUDA / PyTorch / transformers 版本连环冲突
- vLLM 编译永久卡死
- MoE 多卡 RPC 端口冲突
- 模型文件损坏、路径不对、权限不足
- 量化精度选择与质量下降的权衡
- 部署任务与后续任务（如 LoRA 微调）的衔接

这本手册把这些教训整理成结构化、可复用、通俗易懂的知识。
